# 🚀 Quick Start Guide
## Malwa Gift and Toys

## STEP 1: First Time Setup
Double-click: `INSTALL_AND_RUN.bat`
This installs everything and opens your website at http://localhost:3000

## STEP 2: Daily Use (After First Setup)
Double-click: `START_WEBSITE.bat`

## STEP 3: Deploy to Live Website
- For GoDaddy hosting: `DEPLOY_TO_GODADDY_HOSTING.bat`
- For Vercel (recommended): `DEPLOY_TO_VERCEL.bat`

## STEP 4: Connect Domain
See: `docs\GODADDY_SETUP.md`

## Admin Panel
URL: http://localhost:3000 → click "Admin" in navbar
Login: admin@malwa.in / Admin@123
⚠️ Change password before going live!
